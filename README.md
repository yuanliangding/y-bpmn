# y-bpmn-vue

## 概述
由于[bpmn.js](https://bpmn.io/toolkit/bpmn-js/).使用起来复杂。业务中需要对bpmn进行定制化，比如定制自己的ui,定制自己的bpmn元素。需要对bpmn.js相当深度的定制开发。且bpmn.js对新手相对不友好，api封装得比较繁琐。入门学习的门槛比较高。  
本项目旨在简化业务中对bpmn.js的使用。将bpmn.js封装成比较简易好用的vue3 组件
